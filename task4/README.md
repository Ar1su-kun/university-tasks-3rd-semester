# Task 4 -  Realization of some utilities from UNIX
## echo - display a line of text
### options:
1) -n does not display the character '\n' at the end
2) -s does not separate elements
## pwd - output the current working directory
## ls - list directory contents
### options:
1) -R - traverses all directories recursively
2) -l - shows all file attributes
3) -g shows all file attributes except the group name
## cp - copy files
## cmp - compare two files byte by byte
## cat - concatenate files and print on the standard output
### options:
1) -n - Numbers lines continuously
